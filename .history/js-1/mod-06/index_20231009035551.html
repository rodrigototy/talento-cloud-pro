JavaScript I - Módulo 05

Instruções do projeto:

1) Crie um projeto com dois arquivos: index.html e script.js.

2) No arquivo 'index.html' insira apenas a estrutura base HTML e a tag script para conectar o arquivo HTML com o arquivo de extensão JavaScript.

3) Usando os conceitos aprendidos nesse módulo, e sem alterar o arquivo 'index.html', adicione um título simples ao site com o id 'titulo', e um elemento que represente um produto à venda.

3) O produto precisa incluir pelo menos o nome, a descrição e o preço.

4) Pode incluir outros "elementos filhos" se achar necessário como, por exemplo, uma imagem.

5) Procure usar o método simples e o método complexo, ensinados neste tópico.
