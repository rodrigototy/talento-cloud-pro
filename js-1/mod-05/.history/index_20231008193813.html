JavaScript I - Módulo 05

Instruções do projeto:

1) Crie um projeto com dois arquivos: index.html e script.js.

2) No arquivo 'index' insira a estrutura base HTML e dentro da tag 'body' inclua quatro tags vazias: h1, ul, a, ol.

3) Adicione o atributo id="titulo" à tag h1, o atributo href="https://prozeducacao.com.br" à tag 'a', e o atributo id="lista-ordenada" à tag 'ol'.

4) Na sequência, realize a conexão entre o arquivo HTML e o arquivo JavaScript.

5) No arquivo script.js capture os quatro elementos criados, e use a propriedade .innerText para adicionar conteúdo textual aos elementos 'h1' e 'a', e a propriedade .innerHTML para adicionar três itens simples na lista não ordenada, e três itens com links para outros sites na lista ordenada.
